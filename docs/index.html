// 02 · Erste Schritte nach dem Flash

Und jetzt? Vier Handgriffe.

  1. ESP32-S3 spannt einen WLAN-Access-Point namens HP-Remote-Setup auf
  2. Mit diesem WLAN verbinden, dann im Browser 192.168.4.1 öffnen
  3. Dein Heim-WLAN eintragen → das Gerät klinkt sich neu ein
  4. Im Webinterface MQTT, Display-ROIs und Touch einstellen
  5. Home Assistant findet Sensoren und Tasten automatisch per Auto-Discovery
